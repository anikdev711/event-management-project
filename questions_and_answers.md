###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;

console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<p>

#### Answer: A

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>There is a function named sum. This function returns the sum of two variables a and b. Here a = 1 which is a number and b= "2" which is considered as string. So, javascript converts a=1 to string and concatenate with b. As a result the final answer is "12".</i>

</p>
</details>

###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>In the food variable an array of four foods are assigned. Then an object called info is declared. First item of the food which is food[0] is the value of key favoriteFood. Later the value is changed. But it has no effect on the food variable which is initially assigned. As a result food array remain unchanged and only the value of info is changed.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The nums is an array which has four elements. forEach method is used to iterate in nums array. It checks the num is truthy or falsy. If num is 0, the count is not increases because it is falsy. When num is 1 ,2 3 the count increases and final result is 3. </i>

</p>
</details>
